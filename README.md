# Estrutura JSF, Spring, Hibernate (Fase 4)

# Descrição
Estrutura para projetos JSF, incluindo cadastro de usuário utilizando hibernate.

# Requisitos
- Java 7
- Tomcat 8
- Maven 4
 
# Frameworks Utilizados
- JSF 2.2
- Spring 4.1
- Servlet 3.1
- Primefaces 5.2
- Hibernate 5.1

# Import
Importar como maven project.
